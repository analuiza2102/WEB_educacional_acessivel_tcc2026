# WEB Educacional Acessível TCC 2026

Projeto de TCC desenvolvido como uma aplicação web educacional com foco em acessibilidade, inclusão digital e facilidade de acesso a informações sobre vestibulares públicos, cursos gratuitos e materiais de apoio.

## Sobre o projeto

O WEB Educacional Acessível centraliza informações importantes para estudantes que buscam oportunidades educacionais gratuitas. A plataforma foi pensada para oferecer navegação simples, linguagem clara e recursos que favorecem o acesso de diferentes públicos.

## Funcionalidades

- Página inicial com apresentação do projeto e atalhos principais.
- Listagem de vestibulares e detalhes de cada oportunidade.
- Área de cursos gratuitos.
- Banco de provas e materiais de apoio.
- Página sobre o projeto.
- Página de acessibilidade.
- Recurso de alto contraste para melhorar a leitura.
- Navegação com link para pular diretamente ao conteúdo principal.

## Tecnologias utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Radix UI
- Material UI
- Lucide React

## Como executar o projeto

Antes de começar, tenha o Node.js instalado na máquina.

Instale as dependências:

```bash
pnpm install
```

Inicie o servidor de desenvolvimento:

```bash
pnpm dev
```

Depois, acesse no navegador:

```text
http://localhost:5173/
```

## Gerar versão de produção

Para criar a versão final do projeto:

```bash
pnpm build
```

Os arquivos gerados ficam na pasta `dist`.

## Objetivo acadêmico

Este projeto faz parte de um Trabalho de Conclusão de Curso e tem como objetivo propor uma solução web acessível para apoiar estudantes na busca por oportunidades de estudo, vestibulares e cursos gratuitos.
